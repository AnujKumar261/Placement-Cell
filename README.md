# 🚀 Placement Cell Management System

A full-stack web application built using React (frontend), Spring Boot (backend), and MySQL (database).
It provides user registration, login authentication using BCrypt password encryption, and role-based navigation.

## ✨ Features

- 🔐 Secure Login & Registration (BCrypt encryption)
- 👨‍🎓 Role-based access (Admin / Student / Faculty / Coordinator)
- 🗄️ MySQL database integration
- ⚡ REST API using Spring Boot
- 🌐 React frontend (Axios integration)
- 🔄 Dynamic routing with React Router
- 🚀 Fully deployed on cloud

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- React Router DOM
- React Toastify
- Bootstrap

### Backend
- Spring Boot
- Spring Web
- Spring Data JPA
- Spring Security (BCryptPasswordEncoder)
- Hibernate

### Database
- MySQL

### Deployment
- 🟢 Backend: Render
- 🔵 Frontend: Vercel

# ⚙️ How to Run Project


## 🔧 Backend Setup (Spring Boot)
mvn spring-boot:run

Backend runs at:
http://localhost:9090

## 🔧 Frontend Setup 
npm install
npm start

Frontend runs at:
http://localhost:3000
